# Manual simplificado de usuario

## Iniciar sesión

1. Abre `http://localhost:5173`.
2. Ingresa con tu usuario corporativo (cuando se conecte módulo de auth).
3. Verás el dashboard principal de campañas.

## Cargar scripts de call-calling

1. Ve al módulo **Scripts**.
2. Elige campaña.
3. Carga el guion (pasos de conversación).
4. Inicia sesión de llamada guiada.

## Ver leads

1. Ve al módulo **Leads**.
2. Filtra por campaña o fuente (Facebook/Instagram).
3. Abre cada lead para ver datos de contacto.

## Responder llamadas

1. Ve a **Llamadas en vivo**.
2. Acepta o transfiere llamada según política de campaña.
3. Si aplica, sigue el guion en tiempo real.
4. Guarda el resultado de la llamada al finalizar.
